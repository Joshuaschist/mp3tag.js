ID3v2

ID3v2 class

Property
name
Type: string

This always return 'ID3v2'

Property
buffer
Type: ArrayBuffer | Buffer

Saved buffer

Method Static method
isID3v2(buffer)
Type: boolean

Checks if the buffer is tagged with ID3v2

Method
read([offset = 0])
Type: Frame[]

Returns the parsed frames in a tag that's found at the offset of a buffer

Method
validate()
Type: boolean

Checks each frame if its valid otherwise it throws an error

Method
save()
Type: ArrayBuffer | Buffer

Writes the validated tag to the audio. This returns the modified buffer

Method
getFrames()
Type: object

Returns an object containing a property of a frame with its ID as the key. The value is an array of the frame values from each of its frames

Method
addFrame(id, value)
Type: void

Simply pushes a frame. It validates the input first before pushing

Method
removeFrame(id[, index = null])
Type: void

Removes a frame found in a tag. By default, this removes all frames associated with the ID unless if the index was set, the frame found at the index of the associated frames will only be removed

Method
editFrame(id, value[, index = 0, replace = true])
Type: void

Edits a frame in a tag. By default, this edits the first frame (index 0) of the frames associated with the ID. If the old value is an array, the new value will be pushed otherwise the old value will be replaced with the new value

Method
getAudio()
Type: Uint8Array

Returns the raw audio

Options

version
Type: number

Major version to use when saving

padding
Type: number

Number of padding to be added
