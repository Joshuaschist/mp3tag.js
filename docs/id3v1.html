ID3v1

ID3v1 tagger class

Property
name
Type: string

This always return 'ID3v1'

Property
buffer
Type: ArrayBuffer | Buffer

Saved buffer

Property
title
Type: string

Parsed title. When writing, title should not exceed 30 characters

Property
artist
Type: string

Parsed artist. When writing, artist should not exceed 30 characters

Property
album
Type: string

Parsed album. When writing, album should not exceed 30 characters

Property
year
Type: string

Parsed year. When writing, year should not exceed 4 characters

Property
comment
Type: string

Parsed comment. When writing, comment should not exceed 30 characters if track is blank, otherwise 28 characters

Property
track
Type: number

Parsed track. When writing, track should be in range of 1 - 255

Property
genre
Type: number

Parsed genre. When writing, genre should be in range of 0 - 255. For the list of genres, please visit id3.org

Method Static method
isID3v1(buffer)
Type: boolean

Checks if the buffer is tagged with ID3v1

Method
read()
Type: void

Reads the buffer. This sets the ID3v1 properties title, artist, album, year, comment, track, and genre

Method
validate()
Type: boolean

Checks the properties if its a valid ID3v1 tag otherwise it throws an error

Method
save()
Type: ArrayBuffer | Buffer

Writes the tag to the audio. This returns the modified buffer

Method
getAudio()
Type: Uint8Array

Returns the raw audio
