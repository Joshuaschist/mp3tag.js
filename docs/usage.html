Simple Usage
{% capture simple %} ```javascript const buffer = null // Replace this with the mp3 file's buffer const verbose = true // Logs all processes using `console.log` const mp3tag = new MP3Tag(buffer, verbose) // Read the tags from the buffer mp3tag.read() // Handle error if there's any if (mp3tag.errorCode > -1) throw new Error(mp3tag.error) console.log(mp3tag.tags) // Please see `Tags and Frames` mp3tag.tags.title = 'New Title' mp3tag.tags.artist = 'New Artist' mp3tag.tags.album = 'New Album' mp3tag.save() // Saves the new changes ``` {% endcapture %} {{ simple | markdownify }}
Advanced Usage
{% capture advanced %} ```javascript const MP3Tag = require('mp3tag.js') const fs = require('fs') const buffer = fs.readFileSync('/path/to/mp3') const verbose = true // Logs all processes using `console.log` const mp3tag = new MP3Tag(buffer, verbose) // Read the tags from the buffer mp3tag.read({ id3v2: false // Ignore ID3v2 tags when reading }) // Handle error if there's any if (mp3tag.errorCode > -1) throw new Error(mp3tag.error) console.log(mp3tag.tags) mp3tag.tags.title = 'New Title' mp3tag.tags.artist = 'New Artist' mp3tag.tags.album = 'New Album' mp3tag.save({ strict: true, // Strict mode, validates all inputs against the standards. See id3.org // ID3v2 Options id3v2: { padding: 4096, footer: false } }) ``` {% endcapture %} {{ advanced | markdownify }}
