MP3Tag

Exported class

Property
name
Type: string

Name of the library 'MP3Tag'

Property
version
Type: string

Current version of the library. (eg. "0.6.1")

Property Deprecated
frames
Type: Frame[]

This is only accessible if the tagger used is ID3v2. Please see ID3v2 instead

Property
buffer
Type: ArrayBuffer | Buffer

Saved buffer

Property
tagger
Type: ID3v1 | ID3v2

Returns the tagger used

Property
options
Type: object

Options to be sent to the tagger everytime you call the MP3Tag.read() method

Method
constructor(buffer[, options])

This sets the MP3Tag.buffer and MP3Tag.options

Method
read()
Type: ID3v1 | ID3v2

Automatically detects the ID3 tag version of the buffer then reads it. This returns the tagger used to read the buffer.

Method
save()
Type: ArrayBuffer | Buffer

Validate by running the validate method of the tagger then writes them to the buffer. This returns the old buffer
