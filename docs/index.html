Installation

You can install the library by downloading the distributed package and including them on your scripts. This library does not require any more dependencies. Go to Downloads page

If you have npm, you can also install the library by running the code:

npm install --save mp3tag.js@latest

Usage

Browser

          
            const input = document.getElementById('file')
            input.addEventListener('change', function () {
              const reader = new FileReader()
              reader.onload = function () {
                const mp3tag = new MP3Tag(this.result)
                mp3tag.read()
                console.log(mp3tag.frames)
              }

              if (this.files.length > 0) {
                reader.readAsArrayBuffer(this.files[0])
              }
            })

Node.js

          
            const MP3Tag = require('mp3tag.js')
            const fs = require('fs')

            const buffer = fs.readFileSync('/path/to/mp3')
            const mp3tag = new MP3Tag(buffer)
            mp3tag.read()
            console.log(mp3tag.frames)

MP3Tag

const mp3tag = new MP3Tag(buffer[, options])

Properties

name

Type: string

Name of the library 'MP3Tag'

version

Type: string

Current version of the library. (eg. "0.6.1")

frames

Type: FrameObject[]

Array of frames decoded from the buffer.

buffer

Type: ArrayBuffer | Buffer

Buffer

options

Type: Options

Options passed

Methods

constructor(buffer[, options])

Validates buffer type. This sets the MP3Tag.buffer and MP3Tag.options

read()

Type: void

Reads the buffer. This sets the MP3Tag.frames property

save()

Type: ArrayBuffer | Buffer

Validates the MP3Tag.frames, then writes them to the buffer. Returns the old buffer

getFrames()

Type: object

Returns object containing the id and an array of its values from each of its frames

addFrame(id, value)

Type: void

Simply pushes a frame

removeFrame(id[, index = null])

Type: void

Removes the frame found in a tag. By default, this removes all the frames with the ID. If the index was set, the indexed frame will be removed.

editFrame(id, value[, index = 0, replace = true])

Type: void

Edits a frame found in a tag. By default, the first frame found on a tag will be edited, unless index is set other than 0. If the old value is an array, the value is pushed. Otherwise, if replace is true, the old value will be overwritten.

existsFrame(id)

Type: boolean

Checks if a frame with this ID exists

getBlob()

Type: Blob

Returns the Blob object of the buffer

Options

Properties

version

Type: number
Default: Current version otherwise 3

Save buffer with this ID3v2 major version. Valid major versions are 3, 4.

padding

Type: number
Default: 4096

Number of padding

Frame Object

Properties

id

Type: string

ID of the frame

size

Type: number

Size of the frame (excluding the frame header of 10 bytes)

flags

Type: object

Flags set in this frame

value

Type: any

Value of the frame (type depends on the frame specs)
