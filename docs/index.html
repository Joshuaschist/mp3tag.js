Installation

You can install the library by downloading the distributed package and including them on your scripts. This library does not require any more dependencies. Go to Downloads page

If you have npm, you can also install the library by running the code: npm install --save mp3tag.js@latest

MP3Tag

const mp3tag = new MP3Tag(buffer, [options])
Options
  • version<number>: Saves audio with this ID3v2 major version (valid: 3, 4, default: Current version, if none, then 3)
  • padding<number>: Number of null paddings (default: 4096)
Properties
  • name<string>: Always return 'MP3Tag'
  • version<string>: Current version of the library (eg. "0.5.1")
  • frames<FrameObject[]>: Array of frame objects
  • buffer<ArrayBuffer>: Buffer
  • options<object>: Options passed
Methods
  • read(): Read the buffer then sets mp3tag.frames
  • save(): Validates the mp3tag.frames for errors, then writing them to the buffer
  • getBlob(): Returns the Blob object of the buffer

Frames

Properties
  • id: ID of the frame
  • size: Size of the frame (excluding the frame header of 10 bytes)
  • flags: Flags set in this frame
  • value: Value of the frame (the type depends on the frame)
