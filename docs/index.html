Installation

You can install the library by downloading the distributed package and including them on your scripts. This library does not require any more dependencies. Go to Downloads page

The source codes are written as an ES6 module so you can just copy the src folder and include them on your scripts as module.
<script src="/path/to/mp3tag.mjs" type="module">

You can also use the CDN by including this line on your codes:
<script src="https://cdn.jsdelivr.net/npm/mp3tag.js@latest/dist/mp3tag.min.js">

If you have npm, you can install the library by running the code:
npm install --save {{ site.github.project_title }}@latest
