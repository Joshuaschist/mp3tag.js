Features

Open Source

It is an open source project and free for everyone to use! You can view the source codes at GitHub

Easy to use

It is very easy to use and build. It can do basic tags modification such as reading, writing and removing tags!

Standards Compliant

It strictly follows the ID3 Specifications, and validates your input to comply with the standards.
